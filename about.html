What this site does

This site is a simple inventory tracker. Users can add items with quantities, see them displayed in a table, and view the same data in a chart. The inventory persists using browser localStorage, so it stays after refresh. A “Clear All” button removes all items and wipes localStorage.

Custom feature (Option B)

Low stock items (quantity < 10) are highlighted in the chart to make it easy to spot items that need restocking.
